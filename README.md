# Chatbot with NextJs

** Next , tailwindcss and typescript**

Build locally:

**1. Clone Repo**

```bash
git clone https://github.com/ReneKubax/Backend-Ecommerce
```

**2. Install Dependencies**

```bash
npm i
```


**3. Run App**

```bash
npm run dev
```

**4. Use It**

You should be able to start working with the proyect
